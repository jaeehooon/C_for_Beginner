연습문제 4번


1. 배열 aa[3]을 선언했을 때 배열의 주소를 알아내려면 &aa를 사용한다. (X)
    배열의 주소는 &없이 배열의 이름이 곧 주소이다.
2. 포인터 변수에는 주소를 입력해야한다. (O)
3. *p는 p가 가리키는 곳의 실제값이다. (O)
4. 포인터 변수 p가 가리키고 있는 곳의 값을 123으로 바꾸려면 &p = 123;으로 표현한다.(X)
    *p = 123; 으로 표현해야한다.
5. char *p;를 선언한 후 p=100;이라고 사용하면 별 문제가 없다.  (X)
    p 에는 주곳값이 들어가야한다.

6. int a; 선언 시 주소를 사용할 때는 *a라고 쓴다.(X)
    &a

7. int a; 선언 시 &a = 100; 은 맞는 표현이다.(X)
8. int *a; 선언 시 a = 100;은 오류없이 수행된다. (X)
    a에는 주솟값이 들어가야 한다.
9. int *a; 선언 시 *a는 a가 가리키는 곳의 실제 값을 의미한다.(O)
